# ScoreMatch


#### En esta aplicacion android se usan las sigientes tecnologias 
- Retrofit
- Fragments 
- Picaso (para manejo de imagenes) 
- ReciclerView 
- MVVM(viewmodel,livedata)
- Navigation compoenent
- Kotlin  
- Shared Preferences  
- Control de versiones GIT
